# Smart Palika Fullstack Project Documentation

## Overview
This project is a comprehensive civic complaint management system that facilitates the management of complaints from the community.

## Documentation Links
- **Backend Documentation:** [Backend Documentation Link](#)
- **Frontend Documentation:** [Frontend Documentation Link](#)

## Features Overview
- User registration and authentication
- Complaint submission and tracking
- Complaint resolution workflow
- Admin dashboard for managing complaints

## Tech Stack
### Frontend
- React
- Redux
- Material UI

### Backend
- Node.js
- Express
- MongoDB

## Quick Start Instructions
1. Clone the repository:
   ```
   git clone https://github.com/PrasiddhaBhattarai/Smart-Palika-fullstack.git
   cd Smart-Palika-fullstack
   ```
2. Install dependencies for the frontend and backend.
3. Start the server and client.
